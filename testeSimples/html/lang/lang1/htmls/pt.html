Traduzindo com javascript.

O método hasOwnProperty() em JavaScript é utilizado para verificar se um objeto possui uma propriedade específica.

Ele é comumente usado para evitar que propriedades herdadas de protótipos de objetos sejam verificadas. Em outras palavras, hasOwnProperty() retorna true se o objeto tiver a propriedade como sua própria (não herdada), e false caso contrário.
